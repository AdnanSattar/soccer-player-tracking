# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-12-XX

### Added

- Initial release of Soccer Player Tracking system
- YOLOv8-based object detection for players, referees, and ball
- ByteTrack multi-object tracking implementation
- Team assignment based on jersey color clustering
- Ball possession tracking
- Speed and distance calculation for players
- Camera movement estimation and compensation
- Top-down view transformation
- Field boundary filtering to exclude audience
- Model training script with Roboflow dataset support
- Comprehensive documentation and guides

### Features

- Real-time video processing
- Memory-efficient streaming video processing
- Support for custom trained models
- Pre-trained model fallback
- Configurable field boundaries
- Adjustable filtering parameters

### Documentation

- README.md with installation and usage instructions
- TRAINING_GUIDE.md for model training
- FILTERING_GUIDE.md for accuracy improvements
- CONTRIBUTING.md for contributors
- QUICKSTART.md for quick setup
